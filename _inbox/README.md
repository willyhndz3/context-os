# Inbox

Drop raw content here for processing into the knowledge base.

## What to put here

- Call transcripts (for deep extraction)
- Meeting notes
- Customer emails with insights
- Competitor docs
- Any raw content worth capturing

## How to process

Tell Claude: "Process the files in inbox"

Or for a specific file: "Process [filename] into the knowledge base"

## After processing

Files can be deleted or moved to `_archive/` for reference.
