# Synthesis Documents

This folder contains summary documents that pull together atomic concepts from the knowledge base.

## What Goes Here

- Executive summaries of GTM strategy
- Consolidated messaging frameworks
- Quarterly positioning reviews
- Board-ready narratives

## How to Create

Ask Claude: "Create a synthesis document summarizing our [topic] based on the knowledge base"

The system will:
1. Scan relevant atomic concepts
2. Pull verified evidence
3. Create a cohesive narrative with citations
